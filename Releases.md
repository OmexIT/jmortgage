# Release 1.2 #

## New ##
Added [pmi](http://jmortgage.googlecode.com/svn/tags/release-1.2/site/apidocs/jmortgage/pmi/package-summary.html) package.

Added [PMICalculator](http://jmortgage.googlecode.com/svn/tags/release-1.2/site/apidocs/jmortgage/pmi/PMICalculator.html) interface.

Added [DefaultPMICalculator](http://jmortgage.googlecode.com/svn/tags/release-1.2/site/apidocs/jmortgage/pmi/DefaultPMICalculator.html) class.

Added methods to create PMICalculator implementations and DefaultPMICalculator instantiations in [JMortgageFactory](http://jmortgage.googlecode.com/svn/tags/release-1.2/site/apidocs/jmortgage/JMortgageFactory.html).
